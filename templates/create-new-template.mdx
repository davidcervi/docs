## How to Create a Lesson Template

To create a new lesson template, go to the [Templates](https://platform.tutoros.app/lesson-templates) section in the app, and click the `New Lesson Template` button:

![Create New Lesson Template](/images/create-new-template.png)

You will then be brought to the New Lesson Template screen, where you can set the name, description, category, price, duration, payment policy, cancellation policy, etc. for this new template.

## 1. Enter Setup Details

![New Lesson Template 1](/images/new-template-1.png)

The first step is to set the `Name`, `Description`, `Category`, and `Language of Instruction` for this template.

The Category options are:

- Language Learning
- Maths
- Science
- Music

The Language of Instruction is the language in which the lessons will be conducted. For example, if you are teaching Spanish and delivering the lessons in English, select English. 

## 2. Set Price & Duration

![New Lesson Template 2](/images/new-template-2.png)

The second step is to set the `Price` and `Duration` for this template.

The Price is how much the student needs to pay for a lesson from this template.

The Duration is how long each lesson lasts. The duration options are:

- 30 minutes.
- 40 minutes.
- 45 minutes.
- 60 minutes.
- 90 minutes.
- 120 minutes.

## 3. Define Policies

![New Lesson Template 3](/images/new-template-3.png)

### Cancellation Policy üîÅ

The cancellation policy defines the terms under which students can cancel lessons. There are three options:

1. **Flexible**
   - **Cancellation Window:** Students can cancel or reschedule up to 24 hours before the lesson without penalty.
   - **Refund:** Full refund or credit if canceled within the allowed window.
   - **Late Cancellation:** If canceled less than 24 hours before the session, 50% of the lesson cost is charged.

2. **Moderate**
   - **Cancellation Window:** Cancellations allowed up to 48 hours before the session.
   - **Refund:** Full refund if canceled within the allowed window.
   - **Late Cancellation:** 50% charge if canceled between 24-48 hours; full charge if canceled less than 24 hours before.

3. **Strict**
   - **Cancellation Window:** Cancellations allowed up to 72 hours before the session.
   - **Refund:** Full refund if canceled within the allowed window.
   - **Late Cancellation:** No refund if canceled less than 72 hours before the session.

If you want to offer the first lesson for this template at a discount for each new student, you can enter the discount percentage in this field.

Set the discount to 0 for no discount, 100 for a free lesson, or 50 for half off, and so on.

If you have existing students who should not receive a discount on their first lesson, please click the `Upload discount exclusion list` link and upload a .txt file containing each student's email addresses on a new line. Here is an example file:


Finally, click the `Create template` button to create the template.
